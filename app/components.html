icon

  • у кругов класс circle . у квадратов класс square
  • есть 6 размеров. (см ниже) самый большой это класс size_5 самый маленький size_0.по умолчанию без класса модификатора - самый большой = 55px
  • что бы была кликабельность, вместо дива div.circle сделать a.circle
  • какие-то эмблемы(круги. логика другая. сверху набрасывается как бы маска поверх картинки) добавить класс emblem. картинки должны быть квадраты!!!!!
    с размерами - те же классы модификаторы.
    по умолчанию емблемы с внутренней тенью!!!!!
    Эмблемы молгут быть без тени - no_shadow
    эмблемы работают только для circle
  • адаптивные классы на круги и квадраты- всё как у бутстрапа модификаторы size_0 size_md_3 size_lg_4 size_xl_5в самом мелком разрешени size_0 , в xl и до бесконечности size_xl_5
  • если добавить класс .active на круг или квадрат, то будет подсвечивание

slot

  • на slot добавлять классы для соответсвующего оформления: type_danger type_active type_disabled type_warning type_primary
  • добавить класс has_no_text и исчезнет блок с текстом
  • добавить класс has_sub_info и появится блок с цифрами. ширина блока 120пх
  • добавить класс has_close и будет кнопка закрыть
  • добавить класс has_top_alert и будет алерт тот что слева вверху
  • добавить класс has_bottom_alert и будет алерт тот что слева внизу
  • добавить класс has_bottom_right_alert и будет алерт(или эмблема города) тот что справа внизу

                        "type": "slot"
