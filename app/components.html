mini_info

Параметры:

  • класс label — наличие заголовка
  • класс mini_info_square — квадратная иконка
  • класс has_error — ошибка
  • атрибут disabled на теге input для запрета редактирования

{
	"type": "mini_info",
	"name": "db_name",
	"label": "Доход",
	"icon": "/url",
	"url": "/url",
	"value": "2411 (+48453)",
	"editable": false
}

select

  • класс opened открывает сам список дропа
  • класс select_drop_up список дропа вверх
  • класс select_100p список дропа заполняется ни шрину родителя(по умолчанию ширина авто,изза наполнения , и до ширины родителя)

  • у списка дропа, на li может висеть класс active
